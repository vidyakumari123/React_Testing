## Prerequisites

1. **App running** on port **5173** (Vite default):

   ```bash
   npm run dev
   ```

   For heavier parallel loads, TestSprite recommends **`npm run build && npm run preview`** instead of dev mode.

2. **Python 3.10+** with Playwright:

   ```bash
   cd testsprite_tests
   python -m pip install -r requirements.txt
   playwright install chromium
   ```

## Notes

- Scripts use **absolute XPath** locators generated by the agent; prefer **aria-label** selectors if you rewrite tests manually.
- GitHub **rate limits** apply to unauthenticated API calls; repeated runs may throttle.
- After generation, **TC012** initially failed because clearing the input did not reset results; the app now calls **`onClear` → `reset`** from `SearchBar` so clear returns to the empty state (re-run TestSprite or execute `TC012` locally to confirm).
